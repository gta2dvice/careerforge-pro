# CareerForge Pro

AI-Powered ATS Resume Generator & Job Matcher

## Features

### Week 1

* Resume Builder
* Live Resume Preview
* MongoDB Integration
* Resume Storage

### Week 2

* Job Description Analysis
* ATS Score Calculation
* AI Resume Optimization
* Keyword Extraction
* Resume Rewriting

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Context API

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### AI

* Groq API

## Installation

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm run dev
```
